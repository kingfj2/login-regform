# login-regform
showcasing login and registration from using html css and some javascript

## Features
- Login page (index.html): Users must log in first.
- After successful login, redirects to registration page (registration.html).
- Registration page: Allows new users to register with name, email, date of birth, and password.
- Data stored in browser's localStorage (simulates a database).

## How to Run
1. Open index.html in a browser or use a local server (e.g., `python3 -m http.server 8000`).
2. Register a user on registration.html.
3. Then log in on index.html to be redirected back to registration.html.
